# Lenguajes y Compiladores - Actividad 03 (Fase I)

Este repositorio contiene la Fase I del Tema 3: "Lenguajes y Gramáticas Formales" correspondiente a la asignatura **Lenguajes y Compiladores** de la Universidad Nacional Experimental de Guayana (UNEG).

## Contenido del Repositorio

- `Lenguajes y compiladores.pdf`: Informe de investigación detallado estructurado según la guía de evaluación.
- `programa1_derivaciones/`: Código fuente para el modelado de derivaciones (Generación de genoma/figuras mediante gramáticas).
- `programa2_higiene_gramaticas/`: Código fuente relacionado con la corrección de patologías de gramáticas (ambigüedad, recursividad por la izquierda, falta de factorización).
- `programa3_pgn_afd/`: Código fuente del simulador de Autómata Finito Determinístico (AFD) para un subconjunto simplificado de la notación de ajedrez (PGN).
